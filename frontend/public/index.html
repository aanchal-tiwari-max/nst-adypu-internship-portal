⚙ REST API Reference
All endpoints are served from /api
${[ ['GET', '/api/health', 'Health check — returns server status & student count'], ['GET', '/api/stats', 'Dashboard KPIs — placement rate, resume stats, all counts'], ['GET', '/api/students', 'All students — supports ?search, ?status, ?resume, ?batch, ?page, ?limit, ?sortBy, ?sortDir'], ['GET', '/api/students/:uid', 'Single student by User ID'], ['GET', '/api/mentors', 'All mentors with student list & placement stats'], ['GET', '/api/mentors/:name', 'Single mentor with full student list'], ['GET', '/api/companies', 'All companies with placed students'], ['GET', '/api/batches', 'Batch breakdown — Ramanujan, Hopper, Turing'], ['GET', '/api/placed', 'All placed students with company info'], ].map(([method, path, desc]) => `
${method} ${path}
${desc}
`).join('')}
Example: Filter placed students in Turing batch
GET /api/students?status=Placed&batch=Turing&sortBy=name&sortDir=asc&page=1&limit=20
